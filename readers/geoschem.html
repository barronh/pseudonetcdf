GEOS-Chem

GEOS-Chem used the Binary Punch (bpch) file for many years, but now relies on NetCDF. PseudoNetCDF provides readers and convenience functions for both.

  • gcnc: Newer GEOS-Chem NetCDF files

  • bpch: Best Binary Punch reader (chooses bpch1 or bpch2)

  • bpch1: Binary Punch reader (load to mem)

  • bpch2: Binary Punch reader (memory mapped)

  • geos: Raw GEOS met files.

  • flightlogs: Legacy flight log output.
