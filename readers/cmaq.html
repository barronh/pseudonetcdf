CMAQ

CMAQ uses a reader that inherits many methods from PseudoNetCDFFile, but relies on the ioapi_base class to update meta-data.

  • ioapi: CMAQ NetCDF reader.

  • cmaqomidat: OMI input file.

  • griddesc: GRIDDESC reader to make files like CMAQ that are useful as a starting place.
