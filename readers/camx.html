CAMx

CAMx has many readers for inputs, but the uamiv reader covers all the outputs. If you’re using CAMx with NetCDF outputs, you can use the CMAQ reader.

  • New NetCDF Inpput/Output Files:

    • ioapi: NetCDF IOAPI metadata reader, or

    • netcdf PseudoNetCDF.PseudoNetCDF.core._files.netcdf>: Standard NetCDF.

  • Fortran Formatted Outputs:

    • uamiv: CAMx (and UAMIV) format reads

      • average and instantaneous output

      • gridded emission and initial condition input

    • finst: Finescale instantaneous outputs

    • ipr: Integrated Process Rate (ipr) output file

    • irr: Integrated Reaction Rate (ipr) output file

  • Fortran Binary Inputs:

    • cloud_rain: Cloud/rain input file

    • height_pressure: Height/Pressure (zp) file

    • humidity: Humidity input file

    • landuse: Landuse input file

    • lateral_boundary: Lateral boundary condition file

    • point_source: Point source input file

    • temperature: Temperature file

    • vertical_diffusivity: Vertical diffusivity (KV) file.

    • wind: Wind input file.
